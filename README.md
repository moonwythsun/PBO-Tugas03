# Tugas 03 Inheritance - Java OOP

Project ini dibuat menggunakan Java di NetBeans untuk memenuhi tugas mata kuliah Pemrograman Berorientasi Objek (OOP).

## 📌 Deskripsi
Program ini menerapkan konsep:

- Class dan Object
- Constructor
- Method
- Inheritance
- CRUD SQL Simulation
- Superclass dan Subclass

Tema yang digunakan adalah karakter dari game Honkai: Star Rail.

## 🧩 Struktur Class

### Superclass
- `KarakterHSR`

### Subclass
- `Husbando`

## ⚙️ Fitur Program

### Method CRUD
- Create
- Read
- Update
- Delete

### Method Tambahan Subclass

- `statusKarakter()`
